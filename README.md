# DSC CSUSM Website
Open source Developer Student Club (DSC) website created for DSC CSUSM.

## Getting Started
### Prequisties
* Npm

### Installation
1. Change directory to git repository 
   + `cd dsc-csusm-website`
2. Use npm install 
   + `npm install`

### Developement
`npm run dev`

### Deployment
`npm run build`
